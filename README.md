# GustoExpress

GustoExpress is a food delivery service landing page (webdev assignment) featuring:

- **Hero section** with parallax effect
- **Food menu** showcasing popular dishes
- **Animated counters** for key stats
- **Customer testimonials** carousel
- **Restaurant dashboard** with analytics
- **Live order tracking** visual
- **Interactive order form** for quick orders
- **Secure payment options**
- **Enhanced 3D effects** and micro-interactions
- **Responsive design** for all devices

## Getting Started

1. **Clone or download** this repository.
2. Open `index.html` in your web browser.

No build tools or dependencies are required—everything runs in the browser.

## File Structure

- `index.html` — Main HTML file
- `style.css` — Stylesheet for layout, effects, and responsiveness
- `script.js` — JavaScript for interactivity (charts, counters, carousel, etc.)

## Customization

- Update menu items, testimonials, and images in `index.html` as needed.
- Adjust styles in `style.css` for branding.
- Add or modify JavaScript features in `script.js`.
